# Output Format

Covers how the final optimization block is assembled and delivered.

## Output Format

Write the full optimization block as a single Notion-ready markdown block that can be appended to the Content Topic page under an **"SEO & AIO Optimization"** section. Use subheadings for each of the 13 steps. Use tables where the data is tabular (keywords, SERP competitors). Use code blocks for schema markup.

End the block with a one-line summary: "Primary recommendation: [the single most important change to make before publishing]."
