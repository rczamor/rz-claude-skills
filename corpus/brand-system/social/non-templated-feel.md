# Non-Templated Feel

## What it is
Every social graphic should be brand-consistent but feel crafted, not template-generated. The opposite of this rule is "Canva default" — a graphic that uses the platform's pre-built layouts, generic shapes, and stock illustrations. The audience reads templated graphics as low-effort, regardless of the message inside.

## Why it matters
Template aesthetics signal "this is one of many." Crafted aesthetics signal "this is specific to this post." Both can use the same colors and the same fonts; the difference is in the structural choices — the specific composition, the asymmetric layout that responds to the content, the unique diagram that maps to this argument and no other. Riché's brand depends on every post looking like it was made for that post — the audience trusts that the substance matches.

## How to apply
- **Lay out responsively to the content.** Don't pour every post into the same template grid. Let the diagram, headline, and attribution arrange themselves around what the post is actually saying.
- **Vary the composition.** Some graphics are diagram-dominant (90% diagram, 10% headline). Some are headline-dominant (60% headline, 40% supporting visual). Some are split. A consistent palette and type stack hold the brand together; the layout varies by content.
- **No platform pre-builds.** Don't use Canva's "LinkedIn Tech Post Template #4." Don't use Figma community templates. Build from a blank canvas with the palette and type stack.
- **Consistent does not mean identical.** A series of carousel slides should share visual rhythm but not be xerox copies. Each slide solves its own composition problem.
- **One asymmetric anchor per graphic.** A purely centered, symmetric layout reads as template. An asymmetric placement (diagram offset to one side, headline weighted to the other, attribution in a specific corner) reads as crafted.

## Examples
1. **Crafted: a Five-Step Context Flow LinkedIn post.** The diagram dominates the upper two-thirds; the headline sits in the lower-left in Inter SemiBold; "richezamor" attribution in the lower-right at 60% opacity. The composition responds to the diagram's horizontal flow.
2. **Crafted: a Context Quality Framework post.** The 2x2 matrix dominates the right half; the headline and a short caption stack on the left. Asymmetric, intentional, brand-consistent.
3. **Templated (rejected): every post in a Canva "tech-themed" template** with hexagon mesh background, large gray text panel on the left, generic icon on the right, "Connect with us!" footer. Reads as low-effort regardless of content quality.

## Anti-patterns
- "Template = efficiency." Templated workflow is fine internally; templated output isn't. Build crafted graphics fast by reusing the palette/type/icon vocabulary, not the layout.
- Using Canva's "AI tech" template because it ships with the right colors. The colors are right; the layout signals template. Build from blank.
